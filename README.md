# 人事管理系统 java797

## 项目概述

人事管理系统是一款基于Java SpringBoot与Vue的前后端分离系统。该系统致力于优化企业人事管理流程，提高管理效率。

## 技术栈

- **后端技术**
  - Java
  - SpringBoot2
  - MyBatisPlus
  - 数据库：MySQL

- **前端技术**
  - Vue2
  - ElementUI

## 功能模块

### 系统角色

- 管理员
- 员工

### 管理员功能

- 登录
- 个人中心
- 员工管理
- 部门管理
- 员工考勤管理
- 请假申请管理
- 加班申请管理
- 员工工资管理
- 招聘计划管理
- 员工培训管理
- 部门培训管理

### 员工功能

- 登录
- 个人中心
- 员工考勤管理
- 请假申请管理
- 加班申请管理
- 员工工资管理
- 招聘计划管理
- 员工培训管理
- 部门培训管理
- 员工详细管理

- 后端：Java 1.8+
- 数据库：MySQL 5.7+
